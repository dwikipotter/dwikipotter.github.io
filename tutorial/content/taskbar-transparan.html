Cara membuat taskbar windows menjadi transparan tanpa software apapun

Untuk membuat taskbar windows menjadi transparan sangatlah mudah sekali, banyak software diluar sana yang dapat menjadi solusi anda dalam hal ini. Namun kali ini saya akan memberikan tutorial bagaimana cara membuat taskbar Windows transparan tanpa menggunakan software apapun. Sebelum mulai teman2 harus siaapkan bahannya terlebih dahulu.

Untuk bahannya dapat diunduh disini

Jika bahan sudah siap, simak petunjuk berikut:

Petunjuk:

  • file rarnya kemudian extrak. ada 2 file yaitu ON dan OFF
  • click file ON, lalu enter
  • selesai.

    • jika ingin mengembalikan seperti semula, double click file OFF. Tenang saja ini aman, tidak akan mempengaruhi kinerja windows anda. Lagi pun fungsi file off itu adalah untuk mereset kembali registrinya. Jadi tenang saja
